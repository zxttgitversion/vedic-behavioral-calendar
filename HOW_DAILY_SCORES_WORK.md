## 核心概念：三层叠加模型

想象一个人的一天运势像**收音机调频**：

```
📻 收音机调频过程
═════════════════════════════════════════

[底层音乐]
    ↓
大运周期决定了基础频道
(木星年 vs 土星年？)
    ↓
  基础分数:70分
  (这是你这个时期的"底色")

[调节1：音量旋钮]
    ↓
今日月光强度调节音量
(今天月亮在吉位还是凶位？)
    ↓
加减 5-15 分

[调节2：音质按钮]
    ↓
各星位置微调音质清晰度
(太阳有力吗？木星被挡住了吗？)
    ↓
加减 3-10 分

━━━━━━━━━━━━━━━━━━━━━━━━━━━
最终频率 = 今天的整体运势分数
```

## 分数从哪里来？三个来源

### 来源1：命局基础（你的"基因"）

**问题**：这个人生阶段的整体运势如何？

**答案**：查看**大运—小运关系**
- 6比9火星年 → 好运加持 → 基础 95 分 ✨
- 3比11土星年 → 挑战期 → 基础 90 分 ⚔️
- 6比8 → 磨练期 → 基础 60 分 🪨

**类比**：就像你生日那天选择了一个**出生年代**（1980年代 vs 2020年代），这决定了你这整年的"平均运气值"。

```
┌─────────────────────────────────┐
│  你的大小运组合  →  基础分数      │
├─────────────────────────────────┤
│  木星/木星 (1/1)  →  80分        │
│  木星/金星 (5/9)  →  95分 ⭐️    │
│  火星/土星 (6/8)  →  60分 ⛈️    │
│  土星/木星 (2/12) →  65分        │
└─────────────────────────────────┘
```

**答案**：查看**本命月亮 vs 运行月亮**的关系

**重点**：月亮就像你脑子里的"情绪音量"，用这个月光能量来调节基础分数。

```

       + 今天月光很强（×1.3）

       + 但今天月光最危险（×0.5）
  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  → 这部分贡献：60 × 0.5 = 30分 ⚠️
```

### 来源3：行星流年微调（你的"细节打磨"）

**问题**：如果只考虑月光，故事就完整了吗？NO！

**真实场景**：
- 10个月里有 9 个月"好运受阻"警告 → 假信号！
- 原因：仅用月光（Chandra）视角太单一

**解决方案**：五个关键维度的微调

#### 维度定义

| 维度 | 衡量指标 | 代表行星 | 典型分值 |
|:---:|:---|:---:|:---:|
| **心态** | 心情稳定度、直觉敏感度 | 月亮(40%) + 水星(20%) | ±8分 |
| **体能** | 精力、竞争力、体力 | 太阳(40%) + 火星(20%) | ±8分 |
| **事业** | 工作思维、责任感、权威 | 太阳(30%) + 土星(30%) | ±10分 |
| **财运** | 收入机会、资源流入 | 木星(40%) + 金星(20%) | ±10分 |
| **人缘** | 社交吸引力、人脉 | 金星(40%) + 水星(20%) | ±8分 |

#### 微调机制

**第一步：查房位**

每颗行星在今天都有一个"房位"（12宫位之一）：

```
宫位评分表
═════════════════════════════════
1宫（命）      +0.05 分（中立）
2宫（财）      +0.10 分（好）
3宫（兄弟）    +0.20 分（很好）✨
4宫（家宅）    +0.10 分（好）
5宫（子女）     0.00 分（中立）
6宫（困境）    +0.20 分（但含挑战）
7宫（伴侣）     0.00 分（中立）
8宫（死亡）    -0.25 分（很坏）⚠️
9宫（福德）    +0.25 分（最好）⭐️
10宫（事业）   +0.20 分（很好）
11宫（愿望）   +0.30 分（最好）⭐️
12宫（潜意识） -0.15 分（有阻碍）
```

**第二步：检查遮蔽**

关键问题：**这颗好星星今天被坏星星挡住了吗？**

```
常见遮蔽场景
═════════════════════════════════

场景1：木星在11宫（超好！）
       ✓ 正常计分 +0.30 分

场景2：木星在11宫
       × 但土星在5宫堵着它（Vedha规则）
       ✗ 分数清零！ 0 分
       💡 体验：机会看似多，但无法落地

场景3：木星仅在Lagna视角的11宫被堵
       🟡 弱遮蔽：保留在背景，不显黄框
       
场景4：Rahu(虚行星)在那儿
       ✅ 被过滤！无效遮蔽
```

**第三步：检查行星状态**

行星今天强不强？

```
行星强弱评估
═════════════════════════════════

🟢 旺相 (Exalted)
   × 1.25 倍增
   例：木星在巨蟹座 → 比平时强有力
   
🟡 平稳 (Own Sign)
   × 1.10 倍增
   例：木星在射手座 → 在自己地盘
   
🔴 落陷 (Debilitated)
   × 0.60 倍增
   例：木星在摩羯座 → 被克制，能量虚弱
   
⚫ 逆行 R
   × 0.8 (益星) 或 × 1.2 (恶星)
   例：木星逆行 → 益星变弱，反思多
```

**第四步：汇总维度分数**

以"财运"维度为例：

```
财运分数计算过程
═════════════════════════════════

1️⃣ 木星房位评分
   木星在今天在3宫？ → +0.20 分

2️⃣ 木星是否被遮蔽？
   用Chandra视角检查？
   否 → 保留 0.20 分
   是 → 清零为 0 分

3️⃣ 木星状态调整
   ×1.1（自己的星座）
   → 0.20 × 1.1 = 0.22 分

4️⃣ 发挥权重
   财运维度中，木星权重40%
   → 0.22 × 0.4 = 0.088 分(来自木星)

5️⃣ 加上金星
   金星在5宫（+0）× 1.0 × 0.2权重
   → 0.0 分(来自金星)

6️⃣ 维度基础分
   当日月光生成：基础70 × 今日月光倍数

7️⃣ 财运最终分
   = 70 × 1.2 (月光) × (1 + [0.088+0.0] × 2.0放大)
   = 70 × 1.2 × 1.176
   ≈ 99分（见鬼，今天太旺了！）

如果木星被遮蔽：
   = 70 × 1.2 × (1 + 0)
   = 84分（失去了木星加成）
```

## 分数的"黄框"预警：好运受阻

### 什么时候出现黄框？

```
触发条件
═════════════════════════════════════════════

【强遮蔽】= 月亮视角检测到
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✗ 木星应该在11宫 (+0.30分)
  但土星坐在5宫（遮蔽规则）
  → 木星分数清零
  → 显示黄框警告

【弱遮蔽】= 仅上升视角检测到
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✗ 属于背景信息
  ✓ 不显示黄框（太频繁）

【节点遮蔽】= Rahu/Ketu在那儿
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✗ 被过滤！
  ✓ 不是真遮蔽（影子行星无能力）
```

### 黄框文案如何定制？

**打破"一句通用模板"的做法：**

```
被遮蔽的行星 → 专业化文案

太阳被挡 (光芒被掩盖的感受)
→ [好运受阻]：原本的名望上升机会
  因小人或制度干扰而延迟。

月亮被挡 (心情被拨乱的感受)
→ [心态受阻]：情绪和直觉变得模糊，
  容易在决策前产生不必要的焦虑。

火星被挡 (刀锋变钝的感受)
→ [执行受阻]：行动力遭遇软绵绵的阻力，
  切忌因心急而强行发力。

木星被挡 (贵人失联的感受)
→ [好运受阻]：最关键的"神来之笔"被遮挡。
  机会看似很多，但实质性的助力难以落地。

（金星、水星、土星各有定制文案）
```

## 常见问题详解

### Q1: 为什么每个月好日子都集中在前半月？

**A**：月亮轨道周期是 28 天。当运行月亮接近本命月亮时，整体能量就强。

```
月亮周期（28天）
═════════════════════════════════
第1周：月亮从新月往上升
   ↓ 运行月接近本命月
   ↓ 整体能量 ⬆️（容易在前10天爆发好日子）

第2周：月亮到满月（对立）
   ↓ 能量方向性最大
   ↓ 有好有坏（好则极好，坏则极坏）

第3周：月亮从满月往下降
   ↓ 能量 ⬇️（通常是平稳或偏弱）

第4周：月亮接近新月
   ↓ 能量最低
   ↓ 通常是最弱的几天
```

### Q2: 为什么行星逆行后运势会变？

**A**：逆行时，行星的"表现方式"改变了。

```
正常行星（顺行）
= 外向、向外发散、关注外部机会
例：木星顺行 → 拿来主义，主动求机遇

逆行行星（R标记）
= 内向、向内反思、关注内部修为
例：木星逆行 → 沉思型，审视过去机遇

益星逆行 → 能量减弱 (×0.8)
          因为被压制了向外的冲动

恶星逆行 → 能量增强 (×1.2)
          因为内向的压抑反而变成深层思考
```

### Q3: 为什么有时候明知是好日子却事情还是不顺？

**A**：好分数只代表"客观条件好"，不代表"人为决策对"。

```
分数75分 = 说明
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✓ 天时：环境能量助力
✓ 人和：人脉资源可用
✓ 心态：决策思维清晰

但不代表
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✗ 你的执行力（个人修为）
✗ 你的谈判技巧（后天学习）
✗ 对方的态度（他的星盘另算）

类比：天气预报说"今天适合跑步"
     但你没穿好鞋，还是容易扭伤
```

### Q4: Vedha（好运受阻）到底有多严重？

**A**：看遮蔽星和被遮蔽星的重要性。

```
最严重的遮蔽场景
═════════════════════════════════

🔴 木星(财运吉星)被土星(恶星)遮蔽
   影响：这个月的财运机会大打折扣
   症状：谈判泡汤、投资卡壳
   
🟠 月亮(情绪主宰)被火星(冲动恶星)遮蔽
   影响：判断力下降，容易冲动
   症状：会议说错话、决策失误
   
🟡 金星(人缘吉星)被土星遮蔽
   影响：社交吸引力下降
   症状：聚会冷场、合作谈判难


最轻微的遮蔽场景
═════════════════════════════════

🟢 土星(本身就是恶星)被木星遮蔽
   → 坏事被压制 = 转为中性
   
🟢 Rahu/Ketu(虚行星)遮蔽任何人
   → 假遮蔽，被过滤掉
```

## 三句话总结整个算法
